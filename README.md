# KoolTech — Portfolio UI (Next.js + Tailwind CSS)

A modern, dark-themed, responsive portfolio website for KoolTech (HVAC & Appliance Spare Parts). Built with Next.js App Router and Tailwind CSS. Frontend only, no backend.

## Getting Started

1. Install dependencies

```bash
npm install
```

2. Run the development server

```bash
npm run dev
```

3. Open http://localhost:3000 in your browser.

## Features
- Dark, industrial + tech-inspired design
- No sidebar; top navigation on all pages
- Consistent header and footer
- Home: Hero carousel, short About section, Categories preview, Brands grid, CTAs
- Products: Filter tabs (All, AC Parts, AC Gas, AC Oil, Compressors, Refrigeration Parts, Washing Machine Parts) and product grid cards (image + name)
- About and Contact pages
- Mobile-first and fully responsive
- Smooth transitions and subtle animations

## Replace Placeholder Assets
All images use remote placeholders from Unsplash/Pexels/CDNs configured in `next.config.js`. Replace with your brand images as needed.
